## Как проверить MAC-адрес с помощью регулярного выражения
Учитывая строку str, задача состоит в том, чтобы проверить, является ли данная строка допустимым MAC-адресом или нет, используя регулярное выражение.
Действительный MAC-адрес должен удовлетворять следующим условиям:

1. Он должен содержать 12 шестнадцатеричных цифр.
2. Один из способов их представления - сформировать шесть пар символов, разделенных дефисом (-) или двоеточием (:). Например, 01-23-45-67-89-AB является допустимым MAC-адресом.
3. Другой способ их представления - сформировать три группы из четырех шестнадцатеричных цифр, разделенных точками (.). Например, 0123.4567.89AB является допустимым MAC-адресом.

Примеры:
> Входные данные: str = “01-23-45-67-89-AB”;
Вывод: true
Объяснение: 
Данная строка удовлетворяет всем вышеупомянутым условиям. Следовательно, это допустимый MAC-адрес.

> Входные данные: str = “01-23-45-67-89-AH”;
Вывод: false
Объяснение: 
Данная строка содержит ‘H’, за допустимыми шестнадцатеричными цифрами должны следовать буквы из a-f, A-F и 0-9. Следовательно, это недопустимый MAC-адрес.

Подход: Идея состоит в том, чтобы использовать регулярное выражение для решения этой проблемы. Для вычисления ответа можно выполнить следующие шаги.

* Получаем строку.
* Создайте регулярное выражение для проверки допустимого MAC-адреса, как указано ниже:

```
regex = “^([0-9A-Fa-f]{2}[:-]){5}([0- 9A-Fa-f]{2})|([0- 9a-fA-F]{4}\\.[0- 9a-fA-F]{4}\\.[0-9a-fA-F]{4})$”;
```

Где:
* ^ представляет начало строки.
* ([0-9A-Fa-f]{2}[:-]){5} представляет собой пять групп из двух шестнадцатеричных цифр, разделенных дефисами (-) или двоеточиями (:)
* ([0-9A-Fa-f]{2}) представляет собой одну группу из двух шестнадцатеричных цифр.
* | представляет собой или.
* ( обозначает начало группы.
* [0-9a-fA-F]{4} \\. представляет собой первую часть из четырех шестнадцатеричных цифр, разделенных точками (.).
* [0-9a-fA-F]{4} \\. представляет вторую часть из четырех шестнадцатеричных цифр, разделенных точками (.).
* [0-9a-fA-F]{4} представляет собой третью часть из четырех шестнадцатеричных цифр.
* ) представляет собой окончание группы.
* $ представляет собой окончание строки.

```js
// Javascript program to validate
// MAC Address  using Regular Expression
 
// Function to validate the
// MAC_Address  
function isValidMACAddress(str) {
    // Regex to check valid
    // MAC_Address  
    let regex = new RegExp(/^([0-9A-Fa-f]{2}[:-]){5}([0-9A-Fa-f]{2})|([0-9a-fA-F]{4}.[0-9a-fA-F]{4}.[0-9a-fA-F]{4})$/);
 
    // if str 
    // is empty return false
    if (str == null) {
        return "false";
    }
 
    // Return true if the str
    // matched the ReGex
    if (regex.test(str) == true) {
        return "true";
    }
    else {
        return "false";
    }
}
 
// Driver Code
// Test Case 1:
let str1 = "01-23-45-67-89-AB";
console.log(isValidMACAddress(str1));
 
// Test Case 2:
let str2 = "01:23:45:67:89:AB";
console.log(isValidMACAddress(str2));
 
// Test Case 3:
let str3 = "0123.4567.89AB";
console.log(isValidMACAddress(str3));
 
// Test Case 4:
let str4 = "01-23-45-67-89-AH";
console.log(isValidMACAddress(str4));
 
// Test Case 5:
let str5 = "01-23-45-67-AH";
console.log(isValidMACAddress(str5));
```
